# 어떻게 써요!?

1. `main.js`로 가셔서 셋팅값을 변경해주세요.
2. 본 레포지토리를 github pages로 설정해주세요.
3. `https://레포지토리 주소?newbie=true` 로 가주세요.
4. 미스키에서 인증을 해주고 화면에 냐오는 주소를 복사해서 OBS 브라우저 소스에 붙여넣어주세요.
5. 그럼 끝!
